# 📝 議事録フォーマット・ジェネレーター (Minutes Generator)

ブラウザで動く、シンプルで美しい議事録作成ツールです。
入力した内容を自動でフォーマットし、ワンクリックでクリップボードにコピーできます。

## ✨ 特徴
- **リアルタイムプレビュー**: 左側に入力すると、右側に完成形が即座に反映されます。
- **ワンクリックコピー**: ボタン一つで、チャットやメールに貼り付け可能な形式でコピーできます。
- **インストール不要**: `index.html` ファイル一つだけで動作します。
- **モダンなUI**: Tailwind CSSを使用した、見やすく洗練されたデザインです。

## 🚀 使い方
1. フォルダ内の `index.html` をダブルクリックしてブラウザで開きます。
2. 「日付」「参加者」「議題」「決定事項」「ToDo」を入力します。
3. 右側の「クリップボードにコピー」ボタンを押します。
4. Slack、Teams、メールなどに貼り付けて送信します。
